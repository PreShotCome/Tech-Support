# Pixel art tools — GitHub curated collection

> Source: https://github.com/collections/pixel-art-tools
> Pulled into Theo's brain on 2026-05-25.

Not a single repo — a curated collection of pixel-art editors. If
Ian asks Theo for a pixel-art tool recommendation, here's the list
to draw from.

## Desktop editors

- **aseprite/aseprite** — Animated sprite editor & pixel art tool
  (Windows / macOS / Linux). The de-facto industry standard for
  pixel-art game asset work; paid commercial release.
  https://github.com/aseprite/aseprite

- **LibreSprite/LibreSprite** — GPLv2 fork of Aseprite from before
  it went proprietary. Same workflow, open-source forever.
  https://github.com/LibreSprite/LibreSprite

- **Orama-Interactive/Pixelorama** — Open-source pixel-art multitool
  for sprites, tiles, and animations. Built in Godot.
  https://github.com/Orama-Interactive/Pixelorama

- **cloudhead/rx** — Modern and minimalist pixel editor. Vim-like.
  https://github.com/cloudhead/rx

- **PixiEditor/PixiEditor** — Universal Editor for all your 2D needs.
  Windows-focused, .NET-based.
  https://github.com/PixiEditor/PixiEditor

- **Mateusz-Nejman/Pixed** — Pixel-art editor in C# / .NET 9.
  https://github.com/Mateusz-Nejman/Pixed

- **counter185/voidsprite** — Free pixelart editor in C++ / SDL3.
  https://github.com/counter185/voidsprite

## Web editors

- **piskelapp/piskel** — Simple web-based pixel-art / spriting tool.
  Free, has a hosted version at piskelapp.com.
  https://github.com/piskelapp/piskel

- **jvalen/pixel-art-react** — Pixel-art animation/drawing app in
  React.
  https://github.com/jvalen/pixel-art-react

- **lospec/pixel-editor** — Online canvas-based pixel-art tool
  powering Lospec.com.
  https://github.com/lospec/pixel-editor

- **rgab1508/PixelCraft** — Browser pixel-art editor.
  https://github.com/rgab1508/PixelCraft

- **pixa-pics/pixa-pics.github.io** — Open-source web app for
  pixel-art creation.
  https://github.com/pixa-pics/pixa-pics.github.io

- **maierfelix/poxi** — Flat pixel-art editor.
  https://github.com/maierfelix/poxi

- **jackschaedler/goya** — Pixel-art editor in ClojureScript + Om.
  https://github.com/jackschaedler/goya

- **Kully/pixel-paint** — 32×32 sprite editor.
  https://github.com/Kully/pixel-paint

## Programmatic / library

- **gmattie/Data-Pixels** — Create pixel-art programmatically.
  Includes a JS lib and a desktop playground.
  https://github.com/gmattie/Data-Pixels

- **vsmode/pixel8** — Low-res primitives for art and games in React.
  https://github.com/vsmode/pixel8

## Quick picks

If Ian asks "which one should I use?":
- **Just trying it / wants free + good:** Pixelorama or LibreSprite
- **Will pay for the best:** Aseprite
- **Just in the browser, no install:** Piskel or Lospec's editor
- **Programmatic / generative art:** Data-Pixels
